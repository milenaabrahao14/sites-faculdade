Explicação dos Tipos de Posicionamentos Usados

1. position: sticky;

Usado no menu superior para que ele permaneça visível mesmo quando o usuário rola a página.

2. position: relative;

Aplicado no banner principal, permitindo adicionar elementos no interior dele com base na sua própria posição.

3. position: absolute;

Pode ser usado para elementos como etiquetas de “Novo” ou “Promoção” sobre os pôsteres dos filmes.

4. position: fixed;

Utilizado no botão “Contato”, que fica preso no canto da tela mesmo durante a rolagem da página.

5. float;

Utilizado nas duas caixas de exemplo antes dos filmes para demonstrar como elementos podem flutuar lado a lado.

6. display: flex;

Responsável por organizar os cards de filmes lado a lado, adaptando o layout automaticamente.

7. div como contêiner geral

As divs foram usadas para organizar cada área da página: banner, cards, container de filmes, caixas flutuantes etc.
